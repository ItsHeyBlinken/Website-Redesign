# Images Directory

Place all image assets in this directory.

## Required Images

- `drillrig2x.jpg` - Main drilling equipment image (referenced in index.html)

## Image Optimization

For best performance:
- Use WebP format when possible
- Compress images before uploading
- Use appropriate sizes for different screen resolutions
- Consider using srcset for responsive images

## Image Guidelines

- Maintain aspect ratios
- Use descriptive filenames
- Keep file sizes optimized for web (under 500KB when possible)
