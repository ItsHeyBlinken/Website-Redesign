# W.E.S.T. Drilling Website - Option 2

Modern, industrial design option for W.E.S.T. Drilling website.

## Design Characteristics

**Option 2** features a bold, industrial aesthetic with:

- **Dark Industrial Theme**: Charcoal grays, orange accents, dark header/footer
- **Sharp, Geometric Design**: No border-radius, clean lines, left-aligned sections
- **Bold Typography**: Montserrat/Inter font stack with uppercase navigation
- **Accent Lines**: Left-aligned section titles with orange accent lines
- **Card Design**: Sharp corners with left border accent in orange
- **High Contrast**: Dark backgrounds with white text, strong color contrasts

## Key Design Differences from Option 1

- **Color Scheme**: Dark industrial (charcoal/orange) vs. Option 1's blue/orange
- **Typography**: Uppercase navigation, bolder headings, tighter letter spacing
- **Layout**: Left-aligned sections vs. Option 1's centered approach
- **Cards**: Sharp corners with left border accent vs. rounded cards
- **Buttons**: Uppercase, sharp corners, more industrial feel
- **Header**: Dark background with white text vs. Option 1's light header
